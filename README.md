# 🌙 こどもべやからの だいだっしゅつ！

> お金の5つのひみつをさぐれ！楽しみながらお金の基礎が学べる脱出ゲーム

## 🎮 ゲーム概要

夜10時。パパとママに「もう寝なさい！」と部屋に閉じ込められた小4の けんた。
リビングのチョコレートを取りに、こっそり脱出しよう！

ナビキャラの **コイン（ねこ）** と一緒に謎を解きながら、
リベラルアーツ（リベ大）的なお金の知識が自然と身につきます。

## 🚀 遊び方

### ローカルで動かす

```bash
git clone https://github.com/kkp-15/esc-game.git
cd esc-game
# index.html をブラウザで開くだけ！
```

### GitHub Pages で公開する

1. このリポジトリを GitHub にプッシュ
2. Settings → Pages → Source: `main` ブランチ / `/ (root)`
3. `https://kkp-15.github.io/esc-game/` でアクセス！
